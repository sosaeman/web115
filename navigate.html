Course Resources for JavaScript

  • Smarter Way to Learn JS: This is an invaluable tool for memorizing and learning JavaScript, as well as HTML and CSS.
  • Boil An Egg - My Brain is Mush: This is a short video by Professor D.I. that is funny, really makes the point that programming is not easy, and don't be hard on yourself when you don't remember something.
  • Creative BLOQ: This is very good site to get the creative juices flowing.

Other parts of the Site Worth mentioning:

  • CodeAcademy: Has beginner to advanced exercises, I have used it a lot personally.
  • GitHub: A must do for programmers who want to get their code out to hiring managers.
  • Khan Academy: It has timely videos and simple ways of explaining coding.
  • Lynda.com: The site has best instructors and the best ways of explaning all coding topics. It is well worth your time to watch on any technical topic.
